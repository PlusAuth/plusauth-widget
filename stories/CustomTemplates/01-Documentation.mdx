## Custom Templates

You can provide custom templates to inject into widget. Define templates inside the mount element.

Supported template element forms:

- `<template id="pa-...">`
- `<pa-template id="pa-...">`

Template `id` must be prefixed with `pa-`.

Available templates are as following:
```
pa-info-append
pa-content-prepend
pa-content-append
pa-content-footer
pa-footer-body
```

For example:
```html
<div id="widget">
  <template id="pa-content-prepend">
    <h1>My Header</h1>
  </template>
  <template id="pa-content-footer">
    My Footer
  </template>
  <template id="pa-footer-body">
    <a href="#">My Uri</a>
  </template>
</div>

<script type="application/javascript">
  new PlusAuthWidget('#widget', { mode: 'login' }, window.PlusAuth)
</script>
```
