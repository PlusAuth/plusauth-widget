# Registration Page

The Registration page allows new users to create an account. Like the Login page, it is highly customizable via `modeOptions`.

## Basic Usage

To initialize the widget specifically for registration:

```javascript
const auth = new PlusAuthWidget(document.body, {/* field configurations here*/}, window.PlusAuth)
auth.view.mode = 'signup'
```
