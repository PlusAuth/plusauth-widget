# Login Page

The Login page is the primary entry point for users. It supports password-based authentication, passwordless flows (Email/SMS), and social connections.



## Basic Usage

To initialize the widget with the login view:

```javascript
const auth = new PlusAuthWidget(document.body, {/* field configurations here*/}, window.PlusAuth)
auth.view.mode = 'login'
```
