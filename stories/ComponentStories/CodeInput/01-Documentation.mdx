# CodeInput Component

`PCodeInput` is a digit-based input used for OTP and verification code flows.

## Basic Usage

```vue
<PCodeInput
  v-model="code"
  label="Verification Code"
  :size="6"
/>
```

## Behavior Notes

- `modelValue` is a single string value such as `"123456"`
- only numeric characters are accepted
- focus advances automatically as digits are entered
- `size` defines the rendered digit count when the component is created

If you need to switch between different code lengths at runtime, recreate the component or rerender the surrounding widget with the new `size` instead of mutating the existing instance in place.
