# Installation

### From PlusAuth CDN

<JsonKey json={pkg} >
  ```html
  <script src="https://static.plusauth.com/js/@plusauth/widget/v{{version}}/plusauth-widget.min.js" type="application/javascript"></script>
  ```
</JsonKey>

### From Unpkg
<JsonKey json={pkg} >
  ```html
  <script src="https://unpkg.com/@plusauth/plusauth-widget@{{version}}/dist/plusauth-widget.min.js" type="application/javascript"></script>
  ```
</JsonKey>


You can also use PlusAuth widget as a npm library. To do so install it to your project with:

```shell
# with npm
npm install @plusauth/plusauth-widget

# with yarn
yarn add @plusauth/plusauth-widget

# with pnpm
pnpm add @plusauth/plusauth-widget
```

# Usage

After that all we need to initialize the widget like following:

```html
<script >
  new PlusAuthWidget() // will be rendered at document.body
</script>
```

or

```js
import PlusAuthWidget from '@plusauth/plusauth-widget'

new PlusAuthWidget() // will be rendered at document.body
```
