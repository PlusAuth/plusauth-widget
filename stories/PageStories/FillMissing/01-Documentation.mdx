# Fill Missing Information Page

This page renders dynamic form fields that are requested by the server before continuing.

## Basic Usage

```javascript
const auth = new PlusAuthWidget(document.body, {/* configurations here */}, window.PlusAuth)
auth.view.mode = 'fill-missing'
```

## Context Requirements

Provide `context.details.fields` as an array of strings or `{ name, type }` objects.
