# Login Page

The Login page is the main sign-in entry. It supports identifier/password login and optional social connection buttons.

## Basic Usage

```javascript
const auth = new PlusAuthWidget(document.body, {/* field configurations here*/}, window.PlusAuth)
auth.view.mode = 'login'
```

## Optional Configuration

Use `modeOptions.login.fields` to add, reorder, hide, or replace form fields.

Use `context.client.social` to show social provider buttons.
