# MFA Pages

Supported MFA modes in this Storybook section:

`mfa`, `mfa-email`, `mfa-sms`, `mfa-otp`, `mfa-push`, `mfa-webauthn`, `mfa-fv`

When mode is `mfa`, the widget renders the challenge selector.

```javascript
const auth = new PlusAuthWidget(document.body, {/* configurations here */}, window.PlusAuth)
auth.view.mode = 'mfa'
```

## Prompt-Based Views

`mfa-otp` and `mfa-push` can render different prompts based on `context.details`:

```javascript
context: {
  details: {
    dataUrl: '<< QR Code URL >>',
    secret: '<< Secret >>',
    push_code: '61',
    devices: [{ vendor: 'Apple', model: 'iPhone 15' }]
  }
}
```
