# Passwordless Pages

Supported passwordless modes in this Storybook section:

`passwordless-email`, `passwordless-sms`, `passwordless-otp`, `passwordless-push`, `passwordless-webauthn`

To initialize a specific passwordless view:

```javascript
const auth = new PlusAuthWidget(document.body, {/* configurations here */}, window.PlusAuth)
auth.view.mode = 'passwordless-sms'
```

## Prompt-Based Views

`passwordless-otp` and `passwordless-push` can render alternate prompts using `context.details`:

```javascript
context: {
  details: {
    dataUrl: '<< QR Code URL >>',
    secret: '<< Secret >>',
    push_code: '61',
    devices: [{ vendor: 'Apple', model: 'iPhone 15' }]
  }
}
```
