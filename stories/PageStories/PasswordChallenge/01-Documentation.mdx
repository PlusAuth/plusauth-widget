# Password Challenge Page

This page is shown when the user identity is known and only password verification is needed.

## Basic Usage

```javascript
const auth = new PlusAuthWidget(document.body, {/* configurations here */}, window.PlusAuth)
auth.view.mode = 'password-challenge'
```

## Context Requirements

Provide one identifier in `context.details`, such as `email`, `username`, or `phone_number`.
