# Registration Page

The Registration page is used for account creation with the default signup fields.

## Basic Usage

```javascript
const auth = new PlusAuthWidget(document.body, {/* field configurations here*/}, window.PlusAuth)
auth.view.mode = 'signup'
```

## Optional Configuration

Use `modeOptions.signup.fields` to add custom registration fields or reorder existing ones.
